# 1.13 Lua、xLua 与热更新

> 例子：游戏的大机器用 C++ 或 C# 做好后，把经常改的活动、界面和技能规则写在一张可替换的 Lua 说明书里。xLua 像翻译，让 C# 和 Lua 能互相调用。这样改活动时不用把整个游戏重新上架，只下发新脚本就行。

> 一句话：Lua 常被当成可替换的业务脚本，方便手游热更新。
