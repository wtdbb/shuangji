# 8.3 Shader、HLSL 与 GLSL

> 例子：普通 C++ 代码主要让 CPU 做事，Shader 像给 GPU 的工序说明，让成千上万个小工人同时处理顶点、像素和光照。HLSL、GLSL 是不同图形平台常见的 Shader 语言。

> 一句话：Shader 决定模型和像素怎么被画出来，是渲染程序和 TA 的核心词。

---
### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **Shader、HLSL 与 GLSL** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **Shader、HLSL 与 GLSL** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
