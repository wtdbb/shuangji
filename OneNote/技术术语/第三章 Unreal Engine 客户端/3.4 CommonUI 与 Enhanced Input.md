# 3.4 CommonUI 与 Enhanced Input

> 例子：同一个游戏既要支持键盘、手柄又要支持触屏。Enhanced Input 把“跳跃”定义成动作，再分别绑定空格键和手柄 A；CommonUI 像菜单总管，处理多个页面叠在一起时谁接收输入、返回键退到哪里。

> 一句话：Enhanced Input 管输入动作，CommonUI 管跨平台菜单和焦点。
