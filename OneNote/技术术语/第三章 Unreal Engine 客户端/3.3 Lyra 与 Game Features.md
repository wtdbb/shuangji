# 3.3 Lyra 与 Game Features

> 例子：Lyra 像 Epic 提供的一套样板房，不是直接换皮就上线，而是展示大型 UE 项目怎么接 GAS、输入、UI 和多人网络。Game Features 像可插拔活动包，需要某种玩法时装上，不需要时关掉。

> 一句话：Lyra 是 UE 官方大型项目示例，Game Features 用来把玩法做成可插拔模块。

### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **Lyra 与 Game Features** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **Lyra 与 Game Features** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
