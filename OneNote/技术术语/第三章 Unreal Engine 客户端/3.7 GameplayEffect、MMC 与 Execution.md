# 3.7 GameplayEffect、MMC 与 Execution

> 例子：GameplayEffect 像一张药品说明：加多少血、持续多久、能不能叠加。MMC 像按等级和攻击力计算药量；Execution 像结算窗口，一次把攻击、防御、暴击、减伤等复杂公式算完。

> 一句话：Effect 描述效果，MMC 算一个数值，Execution 负责一整次复杂结算。
