# 3.16 UMG、Slate 与 MVVM

> 例子：UMG 像在编辑器里拖控件搭游戏界面；Slate 像更底层的 UI 零件库，做编辑器工具时常见；MVVM 像把“页面长什么样”和“数据从哪里来”分开，血量变了，界面自己跟着更新。

> 一句话：UMG 做游戏 UI，Slate 做底层和工具 UI，MVVM 负责把数据和界面拆开。

---
### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **UMG、Slate 与 MVVM** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **UMG、Slate 与 MVVM** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
