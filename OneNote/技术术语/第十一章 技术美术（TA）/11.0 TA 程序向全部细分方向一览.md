# 11.0 TA 程序向全部细分方向一览

TA（Technical Artist）是美术和程序之间的桥梁岗，整体偏程序。大厂会按方向细分，中小厂一人多岗。

| 方向 | 核心工作 | 关键词 | 稀缺度 |
|------|---------|--------|--------|
| **渲染 TA** | Shader 编写、材质系统、PBR/NPR 风格落地、画面方案 | Shader、HLSL、MaterialInstance、渲染管线 | ⭐⭐⭐⭐ |
| **程序化 TA** | Houdini 批量生成地形/植被/道路，PCG 管线搭建 | Houdini、PCG、HDA、VEX、程序化生成 | ⭐⭐⭐⭐⭐ |
| **管线 TA** | DCC 工具链自动化、Python 脚本、资产导入规范 | Python、Maya 插件、自动化管线、DCC | ⭐⭐⭐⭐ |
| **角色 TA** | 骨骼绑定(Rig)、蒙皮(Skin)、捏脸系统、布料/毛发模拟 | Rig、Skin Weight、Groom、布料模拟 | ⭐⭐⭐⭐ |
| **动画 TA** | Motion Matching、IK 系统、面部捕捉驱动、Control Rig | Motion Matching、IK、面捕、MotionBuilder | ⭐⭐⭐ |
| **特效 TA** | 粒子系统与引擎对接、技能特效表现、VFX 规范制定 | Niagara、Cascade、VFX、粒子数预算 | ⭐⭐⭐ |
| **性能 TA** | DrawCall 优化、LOD 策略、资产规范、包体/内存控制 | DrawCall、LOD、资产规范、性能预算 | ⭐⭐⭐⭐ |

**猎头识别关键信号总表：**

```
渲染TA：Shader / HLSL / MaterialGraph / 渲染管线 / PBR / NPR
程序化TA：Houdini / PCG / HDA / VEX / 程序化生成 / SideFX
管线TA：Python / DCC工具 / 自动化 / Maya插件 / 资产管线
角色TA：Rig / Skin / 绑定 / 蒙皮 / Groom / 布料 / 捏脸
动画TA：Motion Matching / IK / 面捕 / Control Rig / MotionBuilder
特效TA：Niagara / Cascade / VFX / 粒子系统 / 特效规范
性能TA：DrawCall / LOD / HLOD / 资产规范 / 性能优化 / 包体
```

> **稀缺度排名（2026）**：程序化TA（Houdini）> 渲染TA > 性能TA ≈ 管线TA > 角色TA > 动画TA ≈ 特效TA

### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **TA 程序向全部细分方向一览** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **TA 程序向全部细分方向一览** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
