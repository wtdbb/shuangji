# 13.2 常见岗位关键词速查

| 岗位方向 | 简历常见关键词 |
| --- | --- |
| UE 战斗/技能 | GAS、GameplayTag、Montage、CMC、Root Motion、Hitbox、网络预测 |
| UE 系统/玩法 | Gameplay Framework、Lyra、CommonUI、Enhanced Input、UMG、MVVM |
| UE 引擎/底层 | UObject、GC、TaskGraph、Slate、RHI、RDG、异步加载、跨平台 |
| Unity 客户端 | C#、IL2CPP、xLua/HybridCLR、Addressables、UGUI、URP、GC 优化 |
| 游戏服务端 | C++/Go/Erlang、Skynet/ET、Redis、Kafka、K8s、网关、房间、AOI |
| 网络同步 | 帧同步、状态同步、Rollback、预测、校正、延迟补偿、UDP/KCP、DS |
| TA | Shader、材质、Houdini、Python、Maya、Rig、Niagara、资产规范 |
| 测试开发 | 自动化、接口测试、压测、弱网、Gauntlet、兼容性、Crash、性能回归 |

看关键词只能先判断方向，最终仍要问：具体项目、负责模块、个人做了哪部分、遇到什么难点。

### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **常见岗位关键词速查** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **常见岗位关键词速查** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
