# 10.1 Git 与 Perforce（P4）

> 例子：Git 像每个人都带着一份代码时间线，适合分支和合并；P4 像所有人围绕一个中央大仓库工作，大型贴图和场景文件还能先锁住，避免两个人同时改坏。

> 一句话：Git 常见于代码协作，P4 在大型游戏和美术资源管理里很常见。

---
### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **Git 与 Perforce（P4）** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **Git 与 Perforce（P4）** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
