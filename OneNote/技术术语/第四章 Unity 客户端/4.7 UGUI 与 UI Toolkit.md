# 4.7 UGUI 与 UI Toolkit

> 例子：UGUI 像在画布上拖按钮、图片和文本，是游戏运行时界面的主力；UI Toolkit 更像网页，用结构和样式表组织界面，编辑器工具和新项目里越来越常见。

> 一句话：UGUI 偏传统游戏 UI，UI Toolkit 偏结构化界面和编辑器工具。

---
### 💡 猎头应用与常见排雷（补充）
- **简历搜索与匹配**：如果候选人简历中频繁提及 **UGUI 与 UI Toolkit** 相关词汇，说明其参与了对应的技术模块或具备相应底层经验。
- **面试与沟通对齐**：不要仅停留在“是否了解该名词”，重点追问候选人在 **UGUI 与 UI Toolkit** 方面的**实际落地项目**、**遇到过的性能/技术瓶颈**以及**最终的量化成果（如优化的帧率、减少的耗时）**。
- **避坑与定级指南**：许多初中级人员仅停留在“调包”或“在框架内填代码”的阶段。若是推荐大厂高职级或核心引擎组岗，务必核实其是否具备底层机制拆解、架构重构和从0到1搭框架的能力。
