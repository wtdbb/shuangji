# 网易·游戏公司发展史(游策十一)

网易的转身:从门户公司到游戏大厂,再到全球化的考验

## 核心要点
1. **底盘**:《大话/梦幻》回合制 MMO 确立"长线运营+经济系统+自研"模式,2004 年两款占营收 65.5%,是网易后续所有游戏业务的现金流底盘。
2. **跨题材复制**:《阴阳师》(二次元内容)、《第五人格》(强风格)、《蛋仔派对》(年轻化/UGC)证明长线能力可迁移到新用户/新品类。
3. **代理与全球化**:2008 起代理暴雪成为连接全球大作入口;近年自研出海主推《永劫无间》《燕云十六声》《漫威争锋》。
4. **全球化交学费**:2025 年关闭一家美国工作室,收缩加拿大/日本投资;欧美 3A 周期长、成本高、跨文化管理难。
5. **三家对比**:腾讯=平台型(流量/社交/渠道),米哈游=内容工业(技术/美术/二次元全球化),网易=自研+长线运营+经济系统。

今天看网易,很容易把它当成一家天生会做游戏的公司:梦幻西游、阴阳师、永劫无间、燕云十六声,这些名字摆在一起,会让人产生一种错觉——网易好像从一开始就是中国游戏行业的核心玩家。但实际并不是这样。网易最早站上中国互联网舞台时,关键词不是游戏,而是门户、游戏、新闻和上市,和新浪、搜狐一样属于第一代中国互联网公司。那一代公司的共同问题是:有流量、有用户,但缺乏足够稳定的商业模式。对网易来说,游戏不是顺手做出来的副业,而是一次真正改变公司命运的转向——门户解决的是让用户来看,游戏解决的是让用户留下并愿意付费。

《大话西游》(2001)、《梦幻西游》(2004)两款自研 MMO,在短短几年内彻底改变了网易的收入结构:2004 年年报显示两款游戏贡献了总营收的 65.5%。网易由此从"经营流量"的公司,变成"经营长期用户关系和稳定现金流"的公司,并形成自己的游戏经营模型——靠社交系统+经济系统把人长期留住。这套"长线运营"打法成为网易做游戏的底子:不一定每次做最激进玩法,但擅长把产品做成长期生意,稳定现金流让它能养团队、沉淀研发、承受项目不确定性。

进入移动互联网后,用户时间被切碎,短视频/直播/手游冲击 PC 网游逻辑,网易需要证明能力不只绑在回合制/MMO 题材上。《阴阳师》证明它能做年轻化、风格化、角色驱动的二次元内容型产品并长期运营;《第五人格》证明能做强风格、强叙事但非完全大众化的产品;《蛋仔派对》面向新一代年轻用户,低门槛、好友开黑、UGC、短视频传播。网易没有变成另一家公司,而是把最熟悉的长线打法搬到更年轻的用户身上。

另一条线是暴雪:2008 年前后合作,很长时间中国玩家接触魔兽、暗黑等暴雪游戏都绕不开网易,网易由此成为中国玩家连接全球顶级游戏的重要入口,代理涉及本地化、社区、版本节奏、客服、玩家情绪。随后网易想把自己的游戏带到世界——全球化:早年《荒野行动》在日本等海外积累经验,《永劫无间》让海外看到中国团队能做有辨识度的 PC 竞技;《燕云十六声》代表中式文化+开放世界+全球发行,《漫威争锋》借国际 IP+团队竞技争夺海外用户。但全球化不是一帆风顺:网易在海外开设/投资过不少工作室、请了很多欧美大厂背景制作人,可钱能买到团队买不到确定性,欧美 3A 周期长、成本高、跨文化管理复杂。2025 年 Game Developer 报道网易关闭一家美国工作室,同时收缩加拿大和日本投资——这是网易全球化交的一笔学费。

成功模式也带来自身问题:① 长线运营是优势,也容易从"陪伴"变"压力";② 老产品现金流太稳给了耐心也带来路径依赖;③ 全球化尚未完全证明。对比来看:腾讯像平台型公司(流量/社交/渠道/发行),米哈游像内容工业公司(技术/美术/角色/二次元全球化),网易则像"会养游戏的老牌互联网公司",核心能力是自研、长线运营、经济系统和产品矩阵。网易已完成从门户公司到游戏大厂的转身,但能否从中国游戏大厂真正变成全球游戏巨头,是下一阶段要回答的问题。
