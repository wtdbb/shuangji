# 空岛建造（Sky Island / Skyblock）

**品类归属**：[[休闲]] > 建造 / 模拟经营子分支

---

## 一句话

在天空中漂浮的小岛上，用极少的初始资源，通过采集→合成→扩建，逐步建造出完整的空中王国。

---

## 核心玩法循环

```
出生在浮空小岛（资源极少）
  ↓
采集 / 合成基础材料
  ↓
解锁新岛屿 / 扩大地块
  ↓
建设、装饰、开放新系统
  ↓
持续扩张循环
```

核心张力来自「资源稀缺感」和「扩张成就感」的反复循环。

---

## 起源

Minecraft 社区玩家制作的 **Skyblock** 模式：出生点只有一块极小浮空岛+少量物资，靠约束逼出创意，在 YouTube/B站 爆火后独立成品类。

---

## 手游常见变体

| 变体 | 说明 | 代表 |
|------|------|------|
| 空岛 + 放置/挂机 | 岛屿自动产资源，玩家解锁新区域 | 大量超休闲 |
| 空岛 + 模拟经营 | 在浮空岛建城镇，NPC 居民入住 | 部分买量产品 |
| 空岛 + 合成消除 | 两块相同岛合并升级，无尽循环 | Merge 类 |
| 空岛 + 多人 / SLG | 联盟合服，战争资源争夺 | 出海 SLG 变体 |
| 空岛 + 探索叙事 | 飞行探索多个主题岛，有剧情 | 《Sky 光·遇》部分玩法 |

## 猎头关注点

**岗位分布**：策划重、美术轻、程序中等

| 岗位 | 重点 | 说明 |
|------|------|------|
| 关卡策划 | ⭐⭐⭐⭐⭐ | 每个岛屿的资源配置、解锁节奏是核心 |
| 数值策划 | ⭐⭐⭐⭐ | 产出/消耗曲线、付费点设计 |
| 系统策划 | ⭐⭐⭐ | 合成树、解锁树设计 |
| 美术（Low Poly） | ⭐⭐⭐ | 多数空岛游戏用低多边形卡通风 |
| 客户端程序 | ⭐⭐ | 技术要求偏低，Unity 居多 |
| 服务端 | ⭐ | 纯单机放置类几乎不需要 |

**候选人识别信号**：
- 做过「建造」「合成」「放置」「经营」类手游
- 简历有「岛屿」「地块解锁」「扩展玩法」关键词
- 美术候选人有 Low Poly / 卡通 3D 风格作品集

**面试问法**：
- 「你们游戏的核心资源循环是怎么设计的？」
- 「玩家卡关最多的地方在哪，怎么引导的？」
- 「付费点是什么，首日 / 七日留存大概是多少？」

---

## 市场现状（2026）

- 超休闲买量市场依然活跃，空岛建造是出海常见品类之一
- 竞争激烈，同质化严重，差异化靠 IP 或玩法深度
- 国内做这类产品的公司多为中小型，非 A 类客户优先级
