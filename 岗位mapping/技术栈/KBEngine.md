# KBEngine

KBEngine 是一个开源多人在线游戏服务器引擎，主要用于 MMO/MMORPG 或中重度多人在线项目的服务端框架。

## 岗位判断

· 不是 Unity/Unreal 这类客户端引擎，而是“服务器那一侧”的游戏后端框架。

· 简历/岗位里出现 KBEngine，通常说明候选人或项目接触过：多人在线、MMO 服务端、分布式服务器、实体同步、登录/网关/场景服等模块。

· 技术栈倾向：底层 C++，游戏逻辑常用 Python；更偏服务端/架构，不是纯客户端。

## 关联能力

- MMO 服务端
- 分布式游戏服务器
- 玩家登录、连接、断线重连
- 场景/房间/世界服管理
- 玩家、怪物、NPC 等实体系统
- 网络通信与多人同步
- 游戏逻辑热更新

## 面试/筛选口径

看到 KBEngine 可以优先追问：

1. 是改框架底层，还是只写业务逻辑？
2. 做过哪些模块：网关、登录、场景、战斗、AI、同步、数据库？
3. 项目在线规模、同屏人数、分服/合服经验？
4. Python 逻辑层和 C++ 底层分别参与到什么程度？
