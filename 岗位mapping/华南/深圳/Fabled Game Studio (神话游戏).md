# Fabled Game Studio (神话游戏工作室)
标签: #岗位mapping #游戏公司 #深圳

## 一句话简介
位于深圳的高口碑精品独立游戏工作室，由法国制作人Nicolas领衔，以克拉肯（海怪）为标志，专精于高质量的Roguelike卡牌策略游戏开发。

## 旗下游戏
### ✅ 《航海奇闻》 (Pirates Outlaws) — 已上线 (2019)
- 类型：Roguelike卡牌构筑 (Deckbuilder)
- 简介：工作室一战成名的精品代表作，海盗题材肉鸽打牌，曾在Steam和移动端双双取得极高口碑。

### ✅ 《航海奇闻2：传承》 (Pirates Outlaws 2: Heritage) — EA/已上线 (2025底)
- 类型：Roguelike卡牌构筑
- 简介：前作的全面进化续作。
- 数据/报道：2025年12月4日在 Steam 开启抢先体验 (EA)，随后登陆移动端与 Epic。核心创新了“倒计时心跳”、“邻卡连锁共振”、“卡牌同名三合一进化树”以及可自由返航的开放地图探索，策略深度和自由度大幅提升。

### ✅ 《Spiritle》 (灵兽派对) — 已上线
- 类型：4人回合制策略桌游
- 简介：融合了战棋与消除翻牌元素的策略派对游戏。
