国内顶级原创科幻动画公司（《灵笼》出品方），2019年起转型自研游戏，首款自研 ARPG《凡应》研发七年，已于 2025年11月 完成首测。估值 12.7亿（较 2022年25亿缩水近半），腾讯+阅文合计持股 61.82%，已纳入腾讯系。

---

## 主要 IP

| IP | 类型 | 说明 |
|----|------|------|
| 灵笼（Incarnation） | 原创科幻动画 | 代表作，末世生物朋克，全球播放量破亿 |
| 凡应 | 幻想动画 + 自研游戏 | 动游联动，游戏为自研 ARPG |
| 幻镜诺德琳 | 动画 | 奇幻风格 |
| 疯味英雄 | 动画 | 热血向 |

> 注：《灵笼》游戏版由完美世界研发（IP 授权），非艺画开天自研

## 招聘情报

· 游戏团队集中上海，公司总部在武汉
· 核心在研：《凡应》ARPG 全平台（PS5 / XSX / PC / iOS / Android）
· 研发七年，换过至少三任制作人，现已趋于稳定
· 招聘方向：动作战斗、动画表现为主

## 资本动态

· 2025-06：阅文集团斥 3.25亿 收购 26.67% 股权，持股增至 31.48%
· 2026-06：阅文再斥 4亿 追加收购
· B站曾持股，2025年前后有所减持
· 估值：2022年 25亿 → 2025年 12.7亿，缩水近半

## 易混淆：艺画开天 与《三体》

艺画开天**制作了《三体》动画**（中国动画史投资最大项目之一，600+人历时3年），但**不持有《三体》游戏版权**。

| 权利 | 归属 |
|------|------|
| 三体 动画制作 | 艺画开天 ✓ |
| 三体 游戏改编权 | **游族网络**（2020年独家授权） |
| 已过审三体游戏 | 游族旗下《我的三体：2277》（2024年版号） |

> 字节杭州江南工作室也有一个"三体IP游戏"在研，是向游族网络单独购买的授权，和艺画开天无关。
